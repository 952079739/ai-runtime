## 构建决策规则

```
改了什么？
  ├─ *.md / *.rst / *.txt        → 全量构建
  ├─ *.png / *.jpg / *.svg / 图片  → 全量构建
  ├─ *.json / *.yaml / 配置文件     → 全量构建（可能影响语义）
  ├─ 新增/删除目录                  → 全量构建
  └─ 仅 *.py / *.ts / *.go 等代码   → 增量构建
```

**简言之：文档、图片、配置变更 → 全量；纯代码变更 → 增量。**
